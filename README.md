# to-do list App

## Overview
A simple to-do list app built in Python for the internship task.

## Features
It shows a menu.
You can see tasks.
You can add a task.
You can remove a task.
Tasks are saved in a text file called tasks.txt.
Even if you close the program, your tasks will stay saved.

## Usage
Run the following command:
```bash
python todo.py
